Задача:
	Написать интерпретатор для модельного языка, позволяющего создавать роботов для игры на сервере. 
	Программа по заданным в командной строке параметрам ( IP-адрес сервера; Номер порта, на котором запущен сервер; Имя в чате; create - для создания новой игры || join - для того, чтобы присоединиться к существующей игре; Количество игроков в созданной игре (при create)|| Имя игрока, создавшего игру, к которой мы хотим присоединиться, или ее номер (при join); Входной файл с программой на модельном языке; Выходной файл (с логом игры) )
